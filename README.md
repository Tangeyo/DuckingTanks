# DuckingTanks README

## Overview

This game project was developed as part of a software development class, where I had the opportunity to explore Unity.
As a newcomer to Unity and C#, this project served as both a challenge and learning experience.

## Artwork and Music Production

The entire project was developed by scratch by me.
All the artwork for this game was illustrated by me using Procreate. 
The music and sound effects for this game was produced by me using Google Chrome Lab. 

## Development Duration

The entire development process spanned a period of three weeks. 
During this time, I iteratively refined the game mechanics, visuals, and audio components to achieve a cohesive and enjoyable gaming experience.

## Controls

- **Player 1:**
  - Movement: WASD
  - Shoot: E

- **Player 2:**
  - Movement: IJKL
  - Shoot: U

## Gameplay

DuckingTanks is a 2 Player split screen retro themed game, where two tanks shoot rubber ducks to battle it out until defeat. The game was inspired by Nintendo's early 1990s game "Goof Troop".
Some features of this game include powerups, breakable walls and quacking.
The game is unfinished due to time constrants, but still maintains relative functionality.

## How to Run

Download latest stable build for Mac or Windows.
